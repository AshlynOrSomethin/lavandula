# Lavandula

Lavandula color scheme for different apps.

## VS Code Theme

This repository now includes a packaged VS Code theme extension based on the
Ptyxis Lavandula palette.

### Build the `.vsix`

```bash
npm install
npm run package
```

### Install the theme

1. Open VS Code.
2. Run `Extensions: Install from VSIX...`.
3. Select the generated `.vsix` file.
4. Run `Preferences: Color Theme` and choose `Lavandula`.
